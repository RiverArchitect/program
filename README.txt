Description of folder contents:

Naming convention for PPT files with graphical comparisons:
000 - Uncalibrated hydrology (no linkage to MODFLOW, no subsurface ET)
100 - Unimpaired hydrology (linkage to MODFLOW, no pumping, no diversions)
111 - Existing hydrology (linkage to MODFLOW, with pumping, with diversions)

Redwood_Timeseries.zip - Contains 100 and 111 subfolders
- There are 19 timeseries files in each folder for the 19 outlets in Redwood Creek 
- Each file contains simulated streamflow from 10/1/1995 to 9/30/2019
- The last column in each file is the cumulative/routed streamflow to the corresponding outlet
